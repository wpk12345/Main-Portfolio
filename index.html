    LIRI BOT

    Liri Bot was a Rutgers Coding Bootcamp assignment that tasked us with creating a Language Interpretation and Recognition Interface. LIRI is a command line node app that takes in parameters and gives you back data. LIRI will retreive a user's latest tweets, return data on any song via spotify api, and return data on any movie via the OMDB api.
